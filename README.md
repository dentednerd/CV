# joanneimlay.com

My resume, portfolio and blog, built in [Gatsby](https://www.gatsbyjs.org/).

## Installation

```sh
git clone https://github.com/dentednerd/CV.git
cd ./CV
yarn
```

## Usage

```sh
yarn develop
```

In browser, navigate to [localhost:8000](http://localhost:8000).
